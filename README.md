# Events API

REST API для управления мероприятиями.

# Технологии

- .NET 10
- C#
- OpenAPI 
- Swagger UI 

## Запуск

Собрать и запустить проект из папки 
Swagger доступен по адресу: `http://localhost:5161/swagger/index.html`

## Эндпоинты

| Метод  | Путь           | Описание                   | Статусы        |
|--------|----------------|----------------------------|----------------|
| GET    | /events        | Список всех событий        | 200            |
| GET    | /events/{id}   | Событие по ID              | 200, 404       |
| POST   | /events        | Создать событие            | 201, 400       |
| PUT    | /events/{id}   | Обновить событие целиком   | 200, 400, 404  |
| DELETE | /events/{id}   | Удалить событие            | 204, 404       |

## Модель события

```json
{
  "id": "int",
  "title": "string",
  "description": "string",
  "startAt": "2025-06-01T10:00:00",
  "endAt": "2025-06-01T12:00:00"
}
```
